# assets/

Coloque aqui os arquivos de imagem do aplicativo:

| Arquivo              | Uso                              | Tamanho mínimo |
|---------------------|----------------------------------|----------------|
| `icon.png`          | Ícone universal (todas plataformas) | 1024×1024 px |
| `icon_android.png`  | Ícone Android adaptive           | 1024×1024 px   |
| `splash.png`        | Tela de splash                   | 1242×2688 px   |

O `flet build` usa o `icon.png` para gerar automaticamente todos os
tamanhos necessários para cada plataforma.

O arquivo `icon.svg` incluído aqui é apenas um placeholder visual —
**substitua por um PNG real antes de publicar.**
